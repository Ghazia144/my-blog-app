# Blog App

A full-stack blog application where users can create, read, update, and delete posts. Built to practice CRUD operations, backend APIs, and frontend design.

## Features
- User-friendly interface
- Create, Read, Update, Delete (CRUD) functionality
- Responsive UI
- Backend with Node.js and MongoDB
- Optionally, authentication 

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## How to Run Locally
1. Clone the repository: `git clone https://github.com/Ghazia144/my-blog-app.git`
2. Install dependencies: `npm install`
3. Run the server: `node server.js` 
4. Open in browser: `http://localhost:3000`
